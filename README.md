## Preferred utilities

- **Window Manager**: Sway
- **Terminal emulator**: Alacritty
- **Look and feel**:  *(configured with the [nwg-look][1] utility)*
    - **GTK theme**: Nordic
    - **Icon theme**: Vimix dark
    - **Mouse cursor**: Breeze Snow
- **Application launcher**: [bemenu][2]
- **File manager**: Thunar
- **Browser**: Librewolf

[1]: https://github.com/nwg-piotr/nwg-look
